使い方は

Nokogiri に渡す HTML 文字列は <html> から始まる必要はなく、部分的な HTML 文字列でも OK です。

置き換えた文字列を取り出すには、doc.inner_html になります。doc.html だと <html> から始まる文字列になってしまうので注意。
